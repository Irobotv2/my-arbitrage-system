Arbitrage System Script Overview
1. redisinsertion.py

Purpose: Populate Redis database with token and pool data
Key Functions:

Fetch top tokens from Etherscan
Retrieve token info and pool data for Uniswap V2 and V3
Store data in Redis for quick access



2. triggers.py

Purpose: Scan for arbitrage opportunities
Key Functions:

Check for price deviations across pools
Identify liquidity imbalances
Log potential arbitrage opportunities



3. sendtochain.py

Purpose: Execute arbitrage transactions on-chain
Key Functions:

Simulate arbitrage paths
Execute transactions using flash loans
Calculate and report profits



4. liquidityimbalances.py

Purpose: Focused analysis on liquidity imbalances
Key Functions:

Detailed scanning for liquidity imbalances
Simulate and potentially execute imbalance-based arbitrage



5. FlashLoanArbitrage.sol

Purpose: Smart contract for executing arbitrage
Key Functions:

Handle flash loans from Balancer
Execute swaps on Uniswap V2 and V3
Calculate and return profits
